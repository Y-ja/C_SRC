# 📚 C 프로젝트

이 리포지토리는 C 프로젝트의 템플릿입니다. 🎉

## 🚀 시작하기

1. **리포지토리 클론하기**:
    ```bash
    git clone https://github.com/yourusername/c-project-template.git
    ```
2. **디렉토리 이동**:
    ```bash
    cd c-project-template
    ```
3. **빌드하기**:
    ```bash
    make
    ```
4. **실행하기**:
    ```bash
    ./your-executable
    ```

## 📂 프로젝트 구조

- `src/` - 소스 코드
- `include/` - 헤더 파일
- `Makefile` - 빌드 스크립트
